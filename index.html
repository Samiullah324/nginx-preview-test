How it works

From ticket to live preview in three simple steps — powered by Sunset Autopilot and Cursor.

1

Dispatch a ticket

Adopt this repo in Sunset, enable auto-deploy, and dispatch a change — Cursor opens a pull request automatically.

2

Preview deploys

Each PR gets a unique preview URL so you can review changes in a real environment before merging.

3

Merge or close

When the PR is merged or closed, the preview environment is torn down automatically — no cleanup required.
